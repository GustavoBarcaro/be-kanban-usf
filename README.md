# Back End KANBAN USF

Esse readme é util apenas para desenvolvimento :D,

## Requer docker compose v2

no terminal:
```sh
make setup
```
escolha as portas dos serviços
```sh
make run
```
Pronto :)


## Conexão com o banco:
pelo terminal:
```sh
make run
```

## Em aplicativos externos:
- HOST: `seu ip`
- DATABASE: database
- USER: user_name
- PASSWORD: pass123
